Create New School in the System

S.NO Task Screen
1 Create new location (City) of the school Locations → Location
2 Create New Organization Settings → Create Organization
3 Create New School Settings → Create School
4 Create User Settings → Create Users
5 Map School with User Settings → School User Mapping
6 Map School with HQ_ADMIN user Settings → School User Mapping
7 Assign Menu to New Location Settings → Back office Location

Add New Menu

S.NO Task Screen
1 Check the menu for location and existing sequence number Settings → Backoffice Menu Master
2 Create New Menu Settings → Backoffice Menu Master
3 Assign to Location Settings → Backoffice Menu Location
